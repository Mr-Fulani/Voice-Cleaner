# Voice Cleaner - Requirements
# 
# Этот проект использует только стандартную библиотеку Python 3.11+
# Внешние зависимости не требуются.
#
# Все необходимые модули включены в Python:
# - argparse (CLI)
# - logging (логирование)
# - subprocess (выполнение ffmpeg)
# - pathlib (работа с путями)
# - json (парсинг JSON от ffprobe)
# - re (регулярные выражения)
# - os (переменные окружения)
#
# Для работы требуется только:
# - Python 3.11 или выше
# - ffmpeg и ffprobe (устанавливаются через системный пакетный менеджер)

# Если в будущем понадобятся зависимости, добавьте их здесь, например:
# numpy>=1.24.0
# some-package==1.0.0
